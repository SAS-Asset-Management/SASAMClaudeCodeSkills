# 08 — Evidence Capture

## Definition

The extent to which objective evidence is captured alongside condition ratings, including photographs, measurements, instrument readings, and spatial references. Covers photo protocols, measurement standards, tool calibration, positional tagging, and evidence storage.

## Evidence Types

- Photograph capture protocol covering angle, lighting, scale reference, and minimum count
- Measurement standards with tolerances and units
- Tool or instrument calibration records
- Positional, chainage, or asset identifier tagging rules
- Evidence storage location, naming convention, or retention rule

## Maturity Ratings

| Level | Rubric |
| --- | --- |
| 0 | No evidence is captured and only the rating is recorded. |
| 1 | Photos are sometimes taken with no protocol and no spatial tagging. |
| 2 | Some procedures specify evidence but application is inconsistent across asset classes and tool calibration is not routinely recorded. |
| 3 | Every MDR assessment captures defined evidence with a spatial reference and calibration is current and traceable. |
| 4 | Evidence is machine readable, linked to records, and sampled for QA, and deviations trigger reassessment. |
| 5 | Evidence feeds machine assisted assessment, remote review, and longitudinal comparison, and practice is benchmarked against industry. |
