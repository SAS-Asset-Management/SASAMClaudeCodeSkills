## Scoring Rubric

### Fit Score (1–5)

| Score | Criteria |
|-------|----------|
| **5** | Perfect ICP match — exact problem, right size, right vertical and geography |
| **4** | Strong match — most ICP criteria met, high likelihood of need |
| **3** | Moderate match — some criteria met, potential need unconfirmed |
| **2** | Weak match — few criteria met, long-shot opportunity |
| **1** | Poor match — disqualifying criteria present |

### Timing Score (1–5)

| Score | Criteria |
|-------|----------|
| **5** | Active RFP, allocated budget, or contract expiry within 90 days |
| **4** | Recent trigger event, likely buying within 6 months |
| **3** | Strategic alignment, no confirmed timeline (6–12 month nurture) |
| **2** | Potential future need, no current signals (12+ months) |
| **1** | No buying signals detected |
