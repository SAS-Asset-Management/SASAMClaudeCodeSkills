# Onboarding deck — "Working with the Ensemble: 0 → 100"

A self-contained reveal.js training deck that takes a SAS-AM consultant from **0** (no Claude
Code installed) to **100** (has run a full handoff end to end against the sandbox).

Open `index.html` in any browser — no build step.

## Modules

M0 Install · M1 Mental model · M2 The five commands · M3 Writing a packet · M4 On-site ·
M5 Review & collect · M6 Hands-on lab (the "100") · M7 Troubleshooting — plus a consolidated
readiness checklist and close.

## Interactivity

- Keyboard nav (arrows / space), **O** overview, **?** help, **S** speaker notes
- Progress bar + slide numbers (reveal.js built-ins)
- **T** or the ◐ button toggles light/dark (persisted)
- Per-module **checklists are clickable** — tick them through (Enter/Space or click)
- **Copy** buttons on every command block

Brand: SAS-AM navy/green, Source Sans Pro + Source Code Pro, Australian English. Content tracks
the shipped `skills/ensemble/` commands and the packet schema — keep them in lockstep.
