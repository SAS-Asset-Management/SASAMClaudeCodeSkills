# FMECA Context File Template

This template defines the standard structure for context files generated by Workflow 5 (Parse
Document into FMECA Context File). When parsing a technical document, populate each section below
with the information extracted. Delete any section that is entirely inapplicable (not just empty —
if the section could apply but the document didn't contain the information, keep the section header
and note "Not available in source document").

---

## How to Use This Template

1. Copy this template structure into a new file
2. Name the file: `{client}-{asset}-{doctype}-context.md`
3. Populate each section from the source document
4. Add source references throughout: `[Source: Document Name, Section/Page]`
5. Complete the gaps analysis at the end
6. Save to outputs for the user to add to their `references/` directory
