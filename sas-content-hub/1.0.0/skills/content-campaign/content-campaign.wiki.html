SAS Content Hub v1.0.0

Orchestrate the full SAS-AM content pipeline from a single brief — creating a downloadable artefact, gating it behind email capture, publishing a supporting article to Webflow, and promoting it on LinkedIn.

Artefact Creation

Interactive HTML scorecards, checklists, and templates via sas-presentation.

Lead Capture

Email-gated downloads with Cortex4 integration and Cloudflare Turnstile bot protection.

Webflow Publishing

SEO-optimised articles published directly to sas-am.com/resources via CMS API.

LinkedIn Promotion

Brand-voice posts promoting the article to drive traffic and downloads.

Campaign Interview

The orchestrator conducts a 7-question interview before any production begins. Questions are asked one at a time with multiple-choice options.

#QuestionOptions
Q1 Campaign Topic Maintenance strategy, Asset condition, AI/ML, Risk & reliability, Maturity assessment, Custom
Q2 Target Sector Local Government, Water, Resources & Minerals, Transport, Health, Defence, Cross-sector
Q3 Artefact Type Interactive scorecard, Audit checklist, Business case template, Maturity assessment, Technical guide, Custom
Q4 Artefact Details Free text — what it helps the reader do, sections, methodology
Q5 Key Evidence Client outcomes, industry statistics, personal experience, contrarian viewpoints
Q6 CTA Goal Book diagnostic, Request assessment, Download artefact, Contact for proposal, Custom
Q7 Urgency & Timeline Event-tied, Seasonal, Evergreen, Custom

Campaign Brief

After completing the interview, the orchestrator compiles a campaign brief summarising all answers before proceeding to production.

Stage 1: Create Artefact

Invokes /sas-presentation with the artefact brief from Q3–Q4. The output is an interactive HTML file — a scorecard, checklist, template, or other downloadable resource.

  • Presentation type mapped from artefact type selection
  • Topic and sector applied from Q1–Q2
  • Methodology and sections from Q4
  • Output: Interactive HTML file saved to working directory

Gate check: “Does this artefact look right? Any adjustments before we gate it?”

Stage 2: Generate Email Gate

Invokes the email-gate skill to create a branded lead capture form. The form captures name, role, and email, submits to the Cortex4 gate API, and returns a time-limited download URL.

  • Artefact title and description from Q3
  • Resource slug derived from title (kebab-case)
  • Icon type matching artefact category
  • Cloudflare Turnstile bot protection included
  • Output: Self-contained HTML snippet for Webflow embed

Gate check: “Happy with the gate form? The resource slug will be {slug} and leads go to Cortex4.”

Stage 3: Generate Hero Image

Invokes /nano-banana-2 to create a hero image for the article. Uses web search grounding for real-world context and applies the mandatory SAS watermark.

  • Prompt derived from campaign topic and sector
  • Aspect ratio: 16:9 (optimal for Webflow cards)
  • Professional style aligned with SAS-AM brand
  • Watermark applied automatically via post-processing
  • Output: Watermarked hero image (JPG)

Gate check: “Does this hero image work for the article? Want to regenerate or adjust the prompt?”

Stage 4: Write Article

Invokes the webflow-content-creator skill with full campaign context. The article leads with the problem the artefact solves, provides genuine insight, and embeds the email gate within the body.

  • Topic, sector, and evidence from the interview
  • Gate HTML embedded as Webflow Custom Code Embed block
  • SEO-optimised with title tag, meta description, and heading hierarchy
  • Australian English, active voice, SAS-AM brand tone
  • Output: Article content ready for Webflow CMS

Gate check: “Ready to publish, or want to adjust the content?”

Stage 5: Publish to Webflow

Uses webflowPublish.py to create the CMS item and upload the hero image. The article is created as a draft for review before going live.

  • Hero image uploaded to Webflow Assets automatically
  • CMS item created with all required fields populated
  • Draft mode by default — user confirms before publishing live
  • Output: Draft article on Webflow with preview URL

Gate check: “Article is in draft on Webflow. Publish live now, or review in Webflow first?”

Stage 6: Create LinkedIn Post

Invokes the linkedin-post-generator skill to promote the published article. The skill produces all eligible post formats in parallel, then recommends the strongest.

  • Published article URL as the link target
  • Campaign topic and interview material for authentic content
  • CTA: drive traffic to the article (which gates the artefact)
  • Multiple formats evaluated; best recommended
  • Output: LinkedIn post ready to publish

Gate check: “Ready to publish on LinkedIn, or want to switch formats or adjust?”

Email Gate Skill

Generates SAS-AM branded email capture HTML snippets for Webflow Custom Code Embed. The gate pattern captures lead data and sends it to the Cortex4 server, which returns a single-use download URL.

Bot Protection

Cloudflare Turnstile invisible challenge prevents spam submissions.

Lead Storage

Name, role, and email captured and stored in the Cortex4 client database.

Secure Downloads

Time-limited, single-use download URLs prevent link sharing.
