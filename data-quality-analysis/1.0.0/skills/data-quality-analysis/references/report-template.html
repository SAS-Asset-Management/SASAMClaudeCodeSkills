Framework Reference

This assessment was conducted using the Australian Bureau of Statistics Data Quality Framework (ABS Catalogue No. 1520.0, May 2009).

The framework comprises seven dimensions of quality: institutional environment, relevance, timeliness, accuracy, coherence, interpretability, and accessibility.
