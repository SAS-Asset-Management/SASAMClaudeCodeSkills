Tender Assessment Skill v1.0.0

Automated tender discovery, alignment scoring, and pursuit package generation for Victorian government tenders (tenders.vic.gov.au).

Web Scraping

Automated scraping of tenders.vic.gov.au with keyword filtering and new-tender tracking.

Weighted Scoring

100-point scoring matrix across five dimensions for objective tender alignment assessment.

Smart Shortlisting

Automatic categorisation into Shortlisted, Flagged, or Declined with override triggers.

Pursuit Packages

Deep-assessment pursuit packages with win themes, compliance matrices, and team recommendations.

How It Works

The tender assessment workflow follows a five-phase pipeline:

Phase 1 — Scrape
Scrape tenders from tenders.vic.gov.au using keyword filters and new-only flags
Phase 2 — Score
Score alignment using weighted matrix (100 pts total)
Phase 3 — Categorise
Shortlisted ≥80   Flagged 60–79   Declined <60
Phase 4 — Report
Generate shortlist report with scoring breakdown and recommendations
Phase 5 — Pursue
Deep-assess and create pursuit packages for shortlisted tenders

Discovery Process

Before beginning assessment, the skill gathers four required context items:

  • Scraper location — path to the tenders.vic.gov.au scraper tool
  • Keyword filtering — industry/domain keywords for relevance filtering
  • New tenders only flag — whether to process only new (unseen) tenders
  • Output preferences — desired output format (report, package, or HTML)

Skip discovery — if the user provides explicit instructions covering all four context items, proceed directly to Phase 1 without the discovery dialogue.

Phase 1: Scrape

The scraper fetches tenders from tenders.vic.gov.au. Available commands:

Default scrape (all active tenders)

python scraper.py

New tenders only

python scraper.py --new-only

Track new tenders and mark as seen

python scraper.py --track-new

The scraper outputs structured data with the following fields:

Field Type Description
tender_idStringUnique tender identifier from tenders.vic.gov.au
titleStringTender title / name
agencyStringIssuing government agency
categoryStringUNSPSC or agency-defined category
close_dateDateSubmission closing date
descriptionStringFull tender description / scope of work
valueNumberEstimated contract value (if disclosed)
urlURLDirect link to tender on tenders.vic.gov.au

Phase 2: Score

Each tender is scored against a weighted matrix totalling 100 points across five dimensions:

Dimension Weight Assessment Basis
Domain Fit30 ptsHow closely the tender matches SAS-AM core domains (asset management, advisory, analytics)
Industry Match25 ptsAlignment with target industries (rail, transport, infrastructure, utilities)
Service Type20 ptsMatch to SAS-AM service offerings (advisory, audit, benchmarking, analytics)
Strategic Value15 ptsLong-term strategic benefit (reference client, market entry, capability building)
Competitive Position10 ptsLikelihood of competitive advantage (incumbency, relationships, niche expertise)

Total = 100 points — the scoring matrix is designed so that a tender scoring 80+ across all dimensions represents a strong strategic fit warranting pursuit.

Domain Fit (30 pts)

Assesses how closely the tender aligns with SAS-AM core service domains:

ScoreCriteria
30Direct match — asset management, advisory, or analytics engagement
20–29Strong overlap — infrastructure planning, condition assessment, lifecycle strategy
10–19Partial overlap — general consulting with asset management components
1–9Tangential — loosely related domain, would require stretch
0Outside domain — no meaningful connection to SAS-AM capabilities

Industry Match (25 pts)

Measures alignment with SAS-AM target industries:

ScoreCriteria
25Rail / transport — primary target industry with existing track record
18–24Infrastructure / utilities — strong adjacency with transferable expertise
10–17Public sector general — government services, health, education
1–9Other sectors — private sector or non-target government areas
0No industry fit — sector with no relevance or transferable value

Service Type (20 pts)

Evaluates match to SAS-AM service offerings:

ScoreCriteria
20Advisory / consulting — strategic advisory, audit, benchmarking
14–19Analytics / data — data analytics, predictive modelling, decision support
8–13Mixed services — blend of advisory and implementation components
1–7Implementation-heavy — primarily delivery/build with minor advisory
0No service fit — pure construction, IT build, or commodity supply

Strategic Value (15 pts)

Assesses long-term strategic benefit beyond the immediate engagement:

ScoreCriteria
15High strategic value — marquee client, new market entry, or capability showcase
10–14Good strategic value — strengthens existing relationships or builds credentials
5–9Moderate value — standard engagement with some reputational benefit
1–4Limited value — short-term revenue with minimal strategic upside
0No strategic value — one-off engagement with no follow-on potential

Competitive Position (10 pts)

Evaluates SAS-AM likelihood of winning based on competitive dynamics:

ScoreCriteria
10Strong position — incumbent, existing relationship, or niche sole-source
7–9Good position — known to the agency, relevant past performance
4–6Moderate position — competitive field but credible differentiators
1–3Weak position — crowded field, no existing relationship, late entry
0No position — locked out by panel arrangements or mandatory qualifications

Phase 3: Categorise

Based on the total score, each tender is placed into one of three categories:

ScoreCategoryAction
≥80 Shortlisted Proceed to Phase 5 — generate full pursuit package
60–79 Flagged Include in shortlist report for manual review and decision
<60 Declined Log with reason and exclude from further processing

Override Triggers

Automated scoring can be overridden in specific circumstances:

Upgrade to Shortlisted (even if score <80)

  • Tender explicitly requests SAS-AM or names the company
  • Client is a strategic target identified in company profile
  • Sole-source or limited-tender opportunity
  • Directly extends an existing SAS-AM engagement
  • User explicitly requests the tender be shortlisted

Flag for review (even if score ≥80)

  • Closing date is less than 5 business days away
  • Estimated value exceeds typical SAS-AM engagement size by 5x+
  • Requires mandatory qualifications or certifications SAS-AM may not hold
  • Conflicts with an existing client engagement
  • Requires teaming/subcontracting arrangements not yet in place

Phase 4: Shortlist Report

The shortlist report provides a summary of all assessed tenders. It includes six sections:

  1. Executive Summary — total tenders scraped, scored, and categorised
  2. Shortlisted Tenders — full scoring breakdown for each shortlisted tender
  3. Flagged Tenders — tenders requiring manual review with flag reasons
  4. Declined Tenders — summary list with primary decline reasons
  5. Score Distribution — histogram and statistical summary of scores
  6. Recommendations — prioritised pursuit order and resource allocation suggestions

Phase 5: Pursuit Packages

For each shortlisted tender, a comprehensive pursuit package is generated containing:

  1. Tender Summary — key details, dates, value, and issuing agency
  2. Alignment Analysis — detailed scoring rationale for each dimension
  3. Win Themes — 3–5 key differentiators tailored to the tender requirements
  4. Compliance Matrix — mapping of tender requirements to SAS-AM capabilities
  5. Risk Assessment — identified risks and proposed mitigations
  6. Team Recommendations — suggested project team and key personnel
  7. Past Performance — relevant prior engagements and reference projects
  8. Pricing Strategy — recommended pricing approach and competitive positioning
  9. Timeline — response preparation timeline and key milestones
  10. Go/No-Go Recommendation — final recommendation with supporting rationale

Output Formats

The skill supports three output formats:

Shortlist Report Only

Markdown report summarising all tenders with scoring and categorisation. Ideal for quick review cycles.

Full Assessment Package

Shortlist report plus individual pursuit packages for all shortlisted tenders. Complete bid preparation kit.

HTML Report

Self-contained HTML file with interactive scoring views, sortable tables, and print-ready formatting.
