Chart Components Reference

Copy-paste inline SVG patterns for marcov / SAS-AM presentations. All charts are theme-aware, accessible, and interactive.

9. Treemap

Show hierarchical data as nested rectangles proportional to value.

10. Radar / Spider Chart

Compare multiple dimensions across categories. Best for 5-8 axes.

11. Heatmap / Matrix

Display values across two dimensions using colour intensity.

12. Gauge Chart

Display a single KPI value relative to a target or range.

13. Scatter Plot

Show correlation between two variables.

14. Bubble Chart

Like scatter but with a third dimension encoded as bubble size.

17. Quadrant Chart

Classify items into four strategic quadrants (e.g. risk vs impact).

18. Funnel Chart

Show sequential stages with decreasing volumes (e.g. pipeline, process).

19. Sankey Diagram

Show flow and transformation between stages. Width encodes magnitude.

20. Distribution Plot

Show probability density or frequency distribution of a dataset.

Current Portfolio
Target Distribution

21. Violin Plot

Combine box plot with density distribution to show data spread and shape.

22. Circular Bar Chart

Radial bars around a centre point. Effective for comparing a small set of categories with a KPI in the centre.

23. Chord Diagram

Show relationships and flow magnitude between entities in a circular layout.

24. Network Diagram

Visualise relationships between entities. Node size = importance, links = connections.
