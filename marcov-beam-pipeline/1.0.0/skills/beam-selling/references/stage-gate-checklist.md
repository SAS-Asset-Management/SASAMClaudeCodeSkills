# marcov.BEAM — Stage Gate Checklist

Use this checklist to review gate criteria before advancing to the next stage. All criteria must be met to earn the right to advance.

---

## Stage 1: Qualify

**Purpose**: Establish whether a conversation is worth having.
**SPIN Focus**: Situation

| # | Gate Criterion | Status | Evidence | Date |
|---|---------------|--------|----------|------|
| 1 | Problem domain identified | [ ] | | |
| 2 | Access to authority or influence confirmed | [ ] | | |
| 3 | Prospect willing to have diagnostic conversation | [ ] | | |

**Right to Advance**: Prospect has agreed to a discovery conversation.

**If gates unmet — actions to close gaps**:
- No problem domain? → Research the company deeper; look for pain signals in job postings, news, and annual reports
- No access to authority? → Identify a champion or warm introduction path; consider event-based approach
- Not willing to diagnose? → Re-evaluate timing; provide value-add content to build credibility first

---

## Stage 2: Diagnose

**Purpose**: Uncover whether real pain exists and what's driving it.
**SPIN Focus**: Problem → Implication

| # | Gate Criterion | Status | Evidence | Date |
|---|---------------|--------|----------|------|
| 1 | Specific problems articulated by the prospect | [ ] | | |
| 2 | Implications quantified (dollars, time, risk) | [ ] | | |
| 3 | Cost of inaction acknowledged by the prospect | [ ] | | |

**Right to Advance**: Prospect says "This is a problem we need to address."

**If gates unmet — actions to close gaps**:
- Problems not articulated? → Go deeper with Problem questions; share industry benchmarks to prompt comparison
- Implications not quantified? → Use Implication questions to develop severity; bring data from similar organisations
- Cost of inaction not acknowledged? → Present scenario analysis; show what "do nothing" looks like in 12 months

---

## Stage 3: Align

**Purpose**: Confirm shared understanding of the problem and what success looks like.
**SPIN Focus**: Implication → Need-payoff

| # | Gate Criterion | Status | Evidence | Date |
|---|---------------|--------|----------|------|
| 1 | Problem definition agreed by both parties | [ ] | | |
| 2 | Success criteria defined and measurable | [ ] | | |
| 3 | Stakeholder landscape mapped | [ ] | | |
| 4 | Decision-making process understood | [ ] | | |

**Right to Advance**: Prospect asks "What would you recommend?"

**If gates unmet — actions to close gaps**:
- Problem definition not agreed? → Circulate a written summary of findings for sign-off; address any disagreements
- Success criteria not defined? → Propose measurable outcomes based on diagnostic findings; align to their KPIs
- Stakeholders not mapped? → Ask directly: "Who else needs to be involved in this decision?"
- Decision process unclear? → Ask: "Walk me through how a decision like this typically gets made here."

---

## Stage 4: Propose

**Purpose**: Present a tailored recommendation built from diagnostic findings.
**SPIN Focus**: Need-payoff

| # | Gate Criterion | Status | Evidence | Date |
|---|---------------|--------|----------|------|
| 1 | Proposal addresses all stated needs | [ ] | | |
| 2 | Commercial terms clear and understood | [ ] | | |
| 3 | Decision process and timeline confirmed | [ ] | | |
| 4 | Key objections addressed | [ ] | | |

**Right to Advance**: Prospect confirms intent to decide.

**If gates unmet — actions to close gaps**:
- Proposal doesn't address needs? → Revisit diagnostic findings; ensure every recommendation maps to a stated problem
- Commercial terms unclear? → Simplify pricing; offer options (e.g., pilot vs. full engagement)
- Decision process stalled? → Re-engage the economic buyer; understand what's blocking
- Objections unaddressed? → Prepare specific responses; consider bringing in references or case studies

---

## Stage 5: Commit

**Purpose**: Secure formal agreement and transition to delivery.
**SPIN Focus**: Validation

| # | Gate Criterion | Status | Evidence | Date |
|---|---------------|--------|----------|------|
| 1 | Commitment received (verbal or written) | [ ] | | |
| 2 | Commercial terms accepted | [ ] | | |
| 3 | Delivery handover initiated | [ ] | | |

**Right to Advance**: Contract signed.

**If gates unmet — actions to close gaps**:
- No commitment? → Clarify what's holding the decision; address remaining concerns directly
- Terms not accepted? → Negotiate; understand their constraints (budget, timing, scope)
- Handover not started? → Introduce delivery lead; provide transition plan

---

## Stage 6: Deliver & Renew

**Purpose**: Execute, demonstrate value, and earn the right to the next engagement.
**SPIN Focus**: Continuous Discovery

| # | Gate Criterion | Status | Evidence | Date |
|---|---------------|--------|----------|------|
| 1 | Outcomes delivered against success criteria | [ ] | | |
| 2 | Relationship health maintained | [ ] | | |
| 3 | Future needs surfaced | [ ] | | |
| 4 | Case study or reference potential identified | [ ] | | |

**Right to Advance**: New opportunity surfaces → cycle back to Stage 1.

**If gates unmet — actions to close gaps**:
- Outcomes not delivered? → Escalate risks; adjust delivery plan; communicate proactively
- Relationship cold? → Schedule a check-in; bring value (not just status updates)
- No future needs? → Run discovery sessions; share industry trends relevant to their business
- No case study potential? → Focus on delivering measurable outcomes; build the narrative

---

## Closing an Engagement

Not every engagement advances to Stage 6. Legitimate closure outcomes:

| Outcome | When to Use | Action |
|---------|-------------|--------|
| **Closed — Won** | Contract signed, delivery underway | Celebrate; document what worked |
| **Closed — Lost** | Prospect chose a competitor or decided not to proceed | Document reasons; schedule nurture touchpoint in 6 months |
| **Closed — Disqualified** | Prospect doesn't fit ICP or no real pain exists | Document reasons; remove from active pipeline |
| **Closed — Stalled** | No movement for 60+ days despite follow-up | Move to nurture; set a trigger to re-engage |
