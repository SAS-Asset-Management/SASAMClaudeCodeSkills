### Methodology Notes

- **Index calculation**: (Segment conversion rate / Overall conversion rate) x 100. Values >100 indicate above-average performance
- **Lead Quality scoring**: Based on ICP match criteria defined during discovery interview
- **Benchmarks**: Australian B2B LinkedIn Ads benchmarks sourced from LinkedIn Marketing Solutions and industry reports
- **Currency**: All values in AUD unless otherwise stated
- **Attribution**: LinkedIn Campaign Manager uses last-touch attribution within its own ecosystem. Cross-platform comparison uses GA4 last-click attribution

### Glossary

| Term | Definition |
|---|---|
| **CPL** | Cost per Lead — total spend divided by number of leads generated |
| **CTR** | Click-through Rate — clicks divided by impressions |
| **CPC** | Cost per Click — total spend divided by number of clicks |
| **CAPI** | Conversions API — server-side conversion tracking that supplements the Insight Tag |
| **Insight Tag** | LinkedIn's JavaScript pixel installed on websites to track conversions and build retargeting audiences |
| **Lead Gen Form** | LinkedIn's native form format that auto-populates with member profile data |
| **Matched Audience** | Custom audiences built from website visitors, CRM contacts, or account lists |
| **ROAS** | Return on Ad Spend — conversion value divided by spend |
| **View-through conversion** | A conversion that occurs after a user sees (but does not click) an ad |
